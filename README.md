# ADK Google Finance - Projetos de IA

Repositório contendo três abordagens para análise de crédito com LLMs:
1. **credit_score_crewai**: Orquestração de agentes com CrewAI.
2. **credit_score_langchain**: Implementação via LangChain.
3. **credit_score_orchestration**: Controle de fluxo customizado.

Tecnologias: Streamlit, Python, Google Gemini e containerizacao individual com Docker.
